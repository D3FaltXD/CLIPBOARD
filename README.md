# CLIPBOARD
A web based clipboard.

The simplest method for transmitting data across internet-connected devices


<br>

> Current preview is available [here](https://djabrj.gq/).

<br>
<br>

# Features
- Multiple Themes
- Interactive
- Share information among all devices

<br>
<br>

# Techstacks
- PHP
- CSS
- HTML
- SQL
